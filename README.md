# Day 3 - NumPy Basics

This repository contains the solution for the **Day 3 NumPy Basics** assignment from the Data Analyst Training Program. It demonstrates the fundamentals of NumPy, including array creation, indexing, slicing, broadcasting, reshaping, aggregate functions, normalization, and conditional operations.

## Topics Covered

- NumPy arrays
- Random array generation
- Array indexing and slicing
- Broadcasting
- Aggregate functions (`sum`, `mean`, `std`, `min`, `max`)
- Array normalization
- Conditional operations using `np.where()`
- Array reshaping
- Element-wise arithmetic operations

## Tasks Completed

### Main Tasks

- Create a 5×5 random array
- Compute row-wise and column-wise means
- Normalize the array (values between 0 and 1)
- Replace negative values with 0 using `np.where()`

### Supporting Exercises

1. Create a 1D array containing numbers from 1 to 20 and slice the even-indexed elements.
2. Create a 3×3 array and calculate its sum, mean, and standard deviation.
3. Reshape a 1D array of 12 elements into a 3×4 array.
4. Demonstrate broadcasting by adding a scalar and a 1D array to a 2D array.
5. Perform element-wise addition, subtraction, and multiplication on two arrays.
6. Use `np.where()` to classify values above a threshold.
7. Find the maximum and minimum values along with their positions.

## Technologies Used

- NumPy
